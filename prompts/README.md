# Prompts

This folder is waiting on real content from your Lovable chat history. I
can't access Lovable directly, so these need your copy-paste, not a
reconstruction.

## What to fill in

### `master-prompt.md`
The original/foundational prompt that kicked off the build: the first real
instruction you gave Lovable, ideally close to verbatim. If you iterated on
it before the first generation, include that too.

### `evolution.md`
The v1 to current story: what changed in how you prompted between the first
version and what's live now, and why. This is the actual "how I improved it"
narrative for the video. Concrete before/after prompt examples land much
better than a description of the improvement.

Suggested structure per entry:
```
## v1 to v2: [what changed]
**Problem:** what wasn't working
**Old prompt:** [paste]
**New prompt:** [paste]
**Result:** what changed in the output
```

### `design-references.md`
Already drafted below: the sites you pulled visual direction from before
prompting. Add specific links (the exact land-book.com listing, Mobbin shot,
Dribbble/Behance piece) if you remember which ones, since specific beats
general for the video.
